This project was developed independently by Sydnni Ahlheim in 2019. 
It's purpose is to demonstrate Java skills. 
It is a shopping cart application. You can choose the type of item you want to buy, 
then type in the name of the item, the price of the item, and the quantity you want. 
Then you can choose to checkout or continue shopping. Once you are done you can hit 0
to check out and then it displays your total cost and a summary of the items you bought.
